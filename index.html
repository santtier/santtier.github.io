In this project I use the Google Maps JavaScript API Guide. I use JASONP bacause this API uses JASONP. To get and plot directions between two points on the map I use the Google’s direction API To let users search for directions from one place to another I use the Places library with the autocomplete widget. Also the Google Maps API lets you create a directions panel that shows you turn-by-turn instructions so I do it. Lastly there is a feature in Google Maps called waypoints, so if you want to travel to three places, for example, the route created is going to be plotted to all three places

Typeahead

It's like Google's search bar, but only with countries. This project helps users to complete what they are typing. I highlight what the user typed in the list of predictions.
